# 📊 HR Employee Attrition Analysis

## 🔍 Business Problem
The HR department wants to understand why employees
are leaving and identify high-risk groups to design
better retention strategies.

## 🛠️ Tools Used
- **SQL** (MySQL) — data extraction & analysis
- **Power BI** — interactive dashboard
- **Excel** — initial data exploration

## 📁 Dataset
IBM HR Analytics Dataset — 1,470 employees, 35 columns
Source: Kaggle

## 📸 Dashboard Preview
![HR Attrition Dashboard](images/hr_attrition_dashboard.png)

## 🔑 Key Insights
- Overall attrition rate: **16.1%**
- Sales dept has highest attrition: **~21%**
- Overtime employees leave at **2x** the rate
- Employees earning below ₹3K/month most at risk
- Employees with **0-2 years tenure** leave the most
